Random Number Finder is a simple and fun number-guessing game. The goal of the game is to find a randomly generated number within a specific range. Each time the player starts the game, the system secretly generates a random number, and the player must try to guess it.

The player enters a number, and the game gives feedback such as “too high”, “too low”, or “correct number.” Using these hints, the player continues guessing until the correct number is found.

The game helps players practice logic, thinking skills, and number prediction while having fun. It can also show the number of attempts the player used to find the correct number.

This project is created using HTML, CSS, and JavaScript. The interface is colorful and simple so players can easily interact with the game.

Main features:

Random number generation

Player number input

Feedback hints (higher or lower)

Attempt counter

Fun and interactive design
